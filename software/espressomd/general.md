ESPResSo is a highly versatile software package for performing
and analyzing scientific Molecular Dynamics many-particle simulations
of coarse-grained atomistic or bead-spring models
as they are used in soft matter research
in physics, chemistry and molecular biology.
More information on [the ESPResSo homepage](https://espressomd.org/)

## How to use

EspressoMD be accessed by loading the appropriate module.

```bash
module load PDC
module load espressoMD/4.1.4
```

For information on how to run EspressoMD, see [their homepage](https://espressomd.org/wordpress/)
For information on how to submit jobs on Dardel, see
[queueing jobs](../docs/dardel/run_jobs/job_scheduling.md).
